# Travel Companion App

## Overview
This is an Android app developed using Java in Android Studio. The app converts Currency, Fuel, and Temperature values.

## Features
- Currency conversion
- Fuel conversion
- Temperature conversion
- Input validation
- Error handling

## Tools Used
- Android Studio
- Java
- XML

## How to Run
1. Open the project in Android Studio
2. Run it on an emulator or Android phone
3. Select a category
4. Enter the value
5. Click Convert
